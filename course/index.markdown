# 当前研究兴趣
- 针对新型存储技术的编译优化、内存管理运行时优化（内存分配器或者垃圾回收器），及操作系统优化（页面调度、文件系统）。
- 可持久化的内存（主存和辅助存储二合一）。
- 从程序代码自动生成注释和摘要。

## 未来拟开展的研究方向
